myFlix API Documentation

Sl No Request Endpoint Url Method type Query Parameters Request data format Response data format
1 Return a list of all movies /movies GET - - A JSON array holding data about movies
2 Return data about a single movie by title /movies/:title GET /:title - A JSON object holding data about a single movie, containing the title, description, genre, director, image URL, whether it's featured or not.
3 Return data about a genre by name/title /movies/genre/:title GET /:title - A JSON object holding data about a single genre.
4 Return data about a director by name /directors/:name GET /:name - A JSON object holding the name, bio, birthdate and death year of a director.
5 Allow new users to register /users POST - A JSON Object holding data about the new user.

Example:
{
"Username": "user1",
"Password": "password1",
"email": "user1@gmail.com"
}
A JSON Object holding data about the new user created, with an id.

Example:
{
"_id": XXXXX,
"Username": "user1",
"Password": "password1",
"email": "user1@gmail.com"
}
6 Allow users to update their user info /users/:username PUT /:username A JSON Object holding data the user information along with field to update. A JSON Object holding the user's updated data.
7 Allow users to add a movie to their list of favorites /users/:userID/movies POST /:userID - A text message indicating movie has been successfully added.
8 Allow users to remove a movie from their list of favorites /users/:userID/movies/:movieID DELETE /:userID, /:movieID - A text message indicating whether the movie was successfully removed.
9 Allow existing user to deregister /users/:userID DELETE /:userID - A text message indicating whether the user was successfully deleted.
